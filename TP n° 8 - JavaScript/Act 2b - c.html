Casos de uso de Javascript:


Codigo en el body: Cuando se necesita que el codigo se ejecute inmediatamente despues de
cargar el contenido HTML.
- Codigo en el head: Cuando se necesita que el codigo se ejecute
despues de cargar la pagina, o cuando se utilizan funciones que se llaman desde eventos.
- Codigo en archivo externo: Cuando se necesita reutilizar el codigo en varias paginas,
o cuando se quiere separar el codigo Javascript del codigo HTML.
